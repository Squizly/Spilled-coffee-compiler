# Семантические действия для генерации ОПС

## Обозначения

- `□` – пустое действие
- `v` – записать переменную в ОПС
- `k` – записать константу в ОПС
- `+, -, *, /, :=, <, >, ==, !=, <=, >=` – записать операцию в ОПС
- `–'` – записать унарный минус (отличается от бинарного `-`)
- `i` – индексация массива
- `r` – оператор чтения (input)
- `w` – оператор вывода (print)
- `j` – безусловный переход
- `jf` – переход если false
- `m1`, `m2`, `m3` – создать метку (запомнить адрес перехода)

## Таблица семантических действий

| Правило | Действие |
| :--- | :--- |
| `A → Q A` | Сгенерировать код для Q, затем для A |
| `A → λ` | □ |
| `Q → D` | Сгенерировать код для D |
| `Q → E` | Сгенерировать код для E |
| `Q → C` | Сгенерировать код для C |
| `Q → L` | Сгенерировать код для L |
| `Q → I` | Сгенерировать код для I |
| `Q → O` | Сгенерировать код для O |
| `D → Y id M R ;` | Занести id в таблицу переменных (в ОПС не пишется) |
| `Y → int` | Установить тип int |
| `Y → float` | Установить тип float |
| `M → [ num_int ]` | Запомнить размер массива для статического выделения памяти |
| `M → λ` | □ |
| `R → , id M R` | Занести id в таблицу переменных, затем код для M и R |
| `R → λ` | □ |
| `E → id idx = S ;` | v (для id), сгенерировать idx, сгенерировать S, := |
| `idx → [ S ]` | Сгенерировать S, затем i |
| `idx → λ` | □ |
| `S → T U` | Сгенерировать код для T, затем для U |
| `U → + T U` | Сгенерировать код для T, затем + и U |
| `U → - T U` | Сгенерировать код для T, затем - и U |
| `U → λ` | □ |
| `T → F W` | Сгенерировать код для F, затем для W |
| `W → * F W` | Сгенерировать код для F, затем * и W |
| `W → / F W` | Сгенерировать код для F, затем / и W |
| `W → λ` | □ |
| `F → ( S )` | Сгенерировать код для S |
| `F → id idx` | v (для id), сгенерировать idx |
| `F → num_int` | k |
| `F → num_float` | k |
| `F → - F` | Сгенерировать код для F, затем –' |
| `C → if ( V ) { A } B` | Вычислить V, jf m1, сгенерировать A, j m2, m1, сгенерировать B, m2 |
| `B → else { A }` | Сгенерировать код для A |
| `B → λ` | □ |
| `L → while ( V ) { A }` | m1, вычислить V, jf m2, сгенерировать A, j m1, m2 |
| `V → S K S` | Вычислить первый S, вычислить второй S, затем K |
| `K → ==` | == |
| `K → !=` | != |
| `K → <` | < |
| `K → >` | > |
| `K → <=` | <= |
| `K → >=` | >= |
| `I → input ( id idx ) ;` | v (для id), сгенерировать idx, r |
| `O → print ( S ) ;` | Вычислить S, затем w |
